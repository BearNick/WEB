SlayNet one-page site

Что внутри:
- index.html — главная страница
- assets/style.css — стили
- assets/script.js — мелкая интерактивность

Что нужно заменить перед публикацией:
1. @yourusername -> ваш Telegram username
2. hello@slaynet.fun -> ваш рабочий email
3. Тексты в блоках "Ваши боты" / FAQ / CTA под ваши реальные услуги
4. При желании добавить реальные кейсы, цены, отзывы

Публикация:
- Загрузите все файлы на хостинг в корень домена slaynet.fun
- Или сначала проверьте локально, открыв index.html в браузере

Рекомендуемое развитие:
- Добавить портфолио / кейсы
- Подключить форму заявки к Telegram, EmailJS или backend
- Подготовить отдельные офферы под Avito / FL / Freelance.ru


Обновление версии portfolio-accordion:
- В hero-блоке раздел портфолио сделан компактнее через Bootstrap accordion (спойлеры).
- Быстрые переходы вынесены в компактные pill-кнопки.
- Подробности по проектам по-прежнему открываются через modal.
