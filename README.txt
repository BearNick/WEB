SlayNet one-page site

Что внутри:
- index.html — главная страница
- assets/style.css — стили
- assets/script.js — языки и фронтенд-логика формы
- api/contact.php — backend-обработчик формы
- api/contact_lib.php — storage и mail helper-логика
- api/messages.php — защищённый endpoint для просмотра сохранённых заявок
- config/contact.env.example — пример server-side конфига
